# Project-NeoTrace v3

> **An experimental Augmented Reality (AR) computer vision system built with Python.**

Project-NeoTrace v3 is my **first high-end computer vision project** and is currently in an **early development stage**. This project explores the idea of creating a futuristic AR interface capable of recognizing, tracking, and interacting with real-world objects and people through a webcam.

The project combines modern computer vision technologies such as **YOLOv8**, **MediaPipe**, and optional **CLIP AI models** to create an interactive "sci-fi HUD" experience. It also introduces a **Target Memory System**, allowing the application to remember and recognize previously saved objects or faces.

---

## ✨ Features

* 🎯 Persistent object and face tracking
* 🔒 Sci-fi scanning and target lock system
* 🖐️ Hand gesture interaction using MediaPipe
* 🎨 AR drawing and shape recognition
* 🧠 Target Memory System with local persistence
* 🏷️ Optional brand and company identification
* 🖱️ Floating AR tool dock with mouse and gesture controls
* 📷 Screenshot support and interactive HUD

## ⚠️ Disclaimer

No computer vision system is 100% accurate. Recognition and identification systems are confidence-based and may occasionally fail or produce incorrect results. The project prioritizes avoiding false identification whenever possible.
